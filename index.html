My Design Process

Five shades of user centred design

1
  • Discover user needs
  • Conduct interviews, surveys etc. Create experience maps and scenarios to map out the data.
  • Define business needs
  • Work with the product team to define the business needs and how to align them with the user needs. Define KPIs to measure success for important actions.
2
  • Create functional specifications
  • Translate user needs to software features.
3
  • Create conceptual models/mental models
  • Define how a user perceives the product with the help of affordances and signifiers
  • Create work flow diagrams
  • Set the flow of the product

  • Sample

4
  • Navigation design and interafce design
  • Prepare the wireframes

  • Sample

5
  • Color, branding and uniformity
  • Make sure the visual experience is consistent and makes sense
  • High fidelity mockups and prototypes
  • Prepare functional prototypes which can be used for user testing
